# LiteMart POS 功能扩展路线图

## 一、业务功能扩展

### 1.1 会员体系

| 功能         | 说明                       | 优先级 |
| ------------ | -------------------------- | ------ |
| 手机号注册   | 客户通过手机号快速注册会员 | 高     |
| 积分累计     | 消费自动积分，支持积分抵扣 | 中     |
| 会员价       | 会员专属价格，提升复购率   | 中     |
| 消费记录查询 | 会员可查看历史消费明细     | 低     |

**数据库扩展：**

```sql
-- Customer 表
CREATE TABLE "Customer" (
  "id" TEXT PRIMARY KEY,
  "phone" TEXT UNIQUE NOT NULL,
  "name" TEXT,
  "points" INTEGER DEFAULT 0,
  "level" TEXT DEFAULT 'NORMAL',
  "createdAt" DATETIME DEFAULT CURRENT_TIMESTAMP
);
```
