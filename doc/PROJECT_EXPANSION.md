## 三、详细扩展方案

### 3.1 运营 & 营销方向

#### 优惠券系统

- **功能**：满减券、折扣券、限品类券；结账时自动抵扣
- **涉及表**：`Coupon(code, type, discount, minAmount, validFrom, validTo, status)`
- **改造点**：
  - `Order` 表加 `couponId` 字段
  - 结账时校验优惠券有效性并计算折扣
  - 前端加优惠券领取/使用入口

#### 促销活动

- **功能**：限时特价、买二送一、组合套餐
- **涉及表**：`Promotion(name, type, startAt, endAt, conditions, action)`
- **改造点**：结账时匹配促销规则，自动应用优惠

#### 会员积分

- **功能**：消费得积分、积分抵现、积分抽奖
- **涉及表**：`Customer(phone, name, points, level)`, `PointLog`
- **改造点**：
  - 升级 `customerTail` → 真实会员体系
  - 结账时计算积分积累
  - 支持积分抵扣部分金额

#### 消息推送

- **功能**：库存预警、订单通知、营销短信
- **改造点**：
  - 集成邮件服务（nodemailer）或短信网关
  - 定时任务检查低库存并推送
  - 新订单时向管理员发送通知

#### 客户画像

- **功能**：购买频次、热销类目、高价值客户标记
- **改造点**：AI 分析客户行为数据，生成标签和统计

### 3.2 供应链 & 库存方向

#### 供应商 & 采购单（推荐优先做）

- **功能**：供应商管理、采购单创建、到货验收
- **涉及表**：
  ```
  Supplier(id, name, phone, address, status)
  PurchaseOrder(id, supplierId, totalAmount, status, createdAt)
  PurchaseOrderItem(id, purchaseOrderId, productId, quantity, costPrice)
  ```
- **改造点**：
  - 新增供应商管理页面
  - 采购单流程：创建 → 待收货 → 已到货（入库）
  - 到货时自动增加库存
  - 支撑利润统计（采购价）

#### 多仓库

- **功能**：分仓库存、调拨单、跨店查询
- **涉及表**：`Warehouse(id, name, address)`, `Inventory(warehouseId, productId, stock)`
- **改造点**：库存与仓库解耦，支持跨仓调拨

#### 批次管理

- **功能**：生产日期/保质期跟踪、先进先出（FIFO）
- **涉及表**：`Batch(id, productId, batchNo, producedAt, expiredAt, stock)`
- **改造点**：出库时按批次优先扣减，保质期预警

#### 条码管理

- **功能**：商品条码（EAN/UPC）维护、打印标签
- **改造点**：
  - 商品表加 `barcode` 字段
  - 接入条码打印库（如 `jsbarcode`）
  - 扫码枪支持更多格式

#### 库存盘点

- **功能**：定期盘点、盘点差异处理
- **涉及表**：`Stocktake(id, warehouseId, status, createdAt)`
- **改造点**：创建盘点单 → 录入实数 → 系统对比 → 差异调账

- **改造点**：`tests/unit/` 目录

#### PostgreSQL 迁移（多端部署必备）

- **改动**：
  - `prisma/schema.prisma` provider 从 `sqlite` 改为 `postgresql`
  - `prisma/dev.db` 导出数据 → 导入 PostgreSQL
  - 连接字符串改为 `DATABASE_URL` 环境变量
- **收益**：解决 SQLite 并发写入瓶颈，支持多实例部署

#### CI/CD

- **工具**：GitHub Actions
- **流程**：PR → Lint + Test → 构建 → 部署
- **改造点**：`.github/workflows/` 目录

#### 日志系统

- **工具**：Pino（Nitro 内置）或 Winston
- **改造点**：结构化日志、统一日志格式、写入文件或日志服务

#### 监控告警

- **改造点**：

  - 设置库存预警阈值触发告警
  - 错误率监控

#### WebSocket 实时

- **改造点**：

### 3.6 数据分析增强方向

#### 自定义报表

- **功能**：日期区间 + 多维度筛选（品类/收银员/支付方式）
- **改造点**：
  - 查询参数支持日期范围、筛选条件
  - 前端日期选择器 + 多选筛选

#### 需求预测 Agent

- **功能**：基于移动平均预测下周/下月销量
- **实现**：
  - 不依赖外部 API，纯算法实现
  - 历史数据 → 移动平均 → 预测值
  - 输出可视化预测图表
- **改造点**：`server/agents/forecast-agent.ts`

#### 智能补货推荐 Agent（推荐优先做）

- **功能**：基于销售速度预测断货时间，生成采购建议
- **实现**：
  - 计算近 7 天日均销量
  - 预测库存可售天数
  - 断货前 N 天预警 + 建议补货量
- **改造点**：`server/agents/restock-agent.ts`

#### 异常预警 Agent

- **功能**：监控退款率突增、客单价异常波动、库存数据篡改
- **实现**：
  - 定时任务扫描订单/库存异常
  - AI 生成风险事件报告
  - 审计日志异常操作识别
- **改造点**：`server/agents/alert-agent.ts`

#### BI 大屏增强

- **功能**：区域/品类钻取、趋势对比
- **改造点**：
  - 看板支持筛选和下钻
  - 同比/环比数据对比

#### 数据导出增强

- **功能**：PDF/Excel 报告、邮件定时推送
- **改造点**：
